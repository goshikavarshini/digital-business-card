# Digital Business Card Generator

## Overview
A web application that allows users to create a digital business card by entering their details, uploading a profile image, selecting a template, and downloading the card as PNG or PDF.

## Features
- Live Preview
- Profile Image Upload
- Social Media Links
- Three Card Templates
- Download as PNG
- Download as PDF

## Technologies Used
- HTML
- CSS
- JavaScript
- Git
- GitHub
- html2canvas
- jsPDF

## How to Run
- Download or clone the repository.
- Open the project folder in Visual Studio Code.
- Install the Live Server extension (recommended).
- Right-click index.html and select Open with Live Server.
- Fill in the form, upload a profile image, choose a template, and use the download buttons to save the business card as a PNG or PDF.
